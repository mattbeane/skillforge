## Evaluation Criteria

### Passing (Foundation Badge Earned)

| Criterion | Requirement |
|-----------|-------------|
| Calibration detection | Correctly identifies overclaiming/underclaiming |
| Recalibration | Can rewrite claims to match evidence |
| Chain tracing | Identifies links in evidence chains |
| Weak link detection | Correctly identifies where chains break |
| Chain construction | Can build appropriate chains from data |

### Common Mistakes

- Confusing correlation with causation
- Missing selection bias in "overclaim" detection
- Building chains where interpretation exceeds data
- Recalibrating by just adding hedges ("may", "might")
